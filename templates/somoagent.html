Deploy SomoAgent
SomoAgent is a persistent WebSocket RMM agent that connects directly to this dashboard for real-time command delivery, live screenshots, file transfers, and instant telemetry — no polling delay.
1
One-line install (Run as Administrator)
Copy and paste into an elevated PowerShell window. Installs SomoAgent as a SYSTEM scheduled task.
Loading install command...
2
Manual download & install
Download the agent script directly, then run with -Install flag.
Download SomoAgent.ps1
powershell -ExecutionPolicy Bypass -File SomoAgent.ps1 -Install
3
Verify connection
After running the installer, the agent connects via WebSocket within ~15 seconds. You'll see it appear in the Live WebSocket Agents table above with a green WS badge.
4
Uninstall
powershell -ExecutionPolicy Bypass -File "C:\ProgramData\SomoAgent\SomoAgent.ps1" -Uninstall
SomoAgent vs SomTechs-Agent (HTTP Polling)
Both agents work alongside each other. Existing HTTP polling agents continue to work. SomoAgent adds real-time capabilities.
