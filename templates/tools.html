Ubuntu 22.04 VM 8GB RAM recommended 100GB disk

Run the Restic REST server on Proxmox instead of the main box. Snapshots stream directly to Proxmox storage, zero load on your main NAS.

Ubuntu 22.04 VM 2GB RAM Large disk / NFS mount
